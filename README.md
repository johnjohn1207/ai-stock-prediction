# AI 股價預測與多因子量化回測系統

本專案是一個基於 LSTM 深度學習模型的股價預測與回測系統，近期已重構為具備「資料存取、邏輯運算、應用展示」的三層式模組化架構，以提升系統擴充性與回測嚴謹度。

系統使用 Streamlit 製作互動式網頁介面，並結合 yfinance 取得跨市場歷史股價資料。除了 AI 動能預測外，更結合了布林通道 (Bollinger Bands) 與三維度多因子過濾機制。用戶可自訂參數，系統將預測未來股價漲跌並模擬真實交易策略績效。

部署網址：https://ai-stock-prediction-buknzvrsdxybkhmr9ijkjv.streamlit.app/

## 主要功能

* 股票資料自動下載（yfinance，支援台/美股）
* **模組化系統架構**（資料層、模型層、回測層完全解耦分離）
* LSTM 模型訓練與預測（具備嚴謹的無未來函數資料切分）
* **多因子交集過濾**（整合 AI 預測、順勢均線與量價表態條件）
* 支援自訂訓練與風控參數（視窗天數、訓練輪數、停損/停利等）
* 預測明日漲跌幅與目標收盤價，並提供明日操作建議
* 回測 AI 策略與大盤（Buy & Hold）累積收益率比較
* 顯示多項專業量化績效指標（RMSE、方向準確度、夏普比率、最大回撤、勝率等）
* 交易紀錄明細表

## 使用方法

1. 安裝依賴：`pip install -r requirements.txt`
2. 啟動網頁：`streamlit run app.py`
3. 於網頁介面輸入股票代號與參數，點擊「執行模型訓練與回測」即可。

## 主要參數說明

* **股票代號**：如 `2330.TW`、`AAPL`
* **滑動視窗天數（Look Back）**：用於 LSTM 的歷史資料長度
* **訓練輪數（Epochs）**：模型訓練次數
* **初始本金**：模擬交易的起始資金
* **停損/停利比例**：強制出場的風險控管條件

## 檔案說明

針對系統維護與未來擴充，本專案將原本單一程式碼拆分為以下核心模組：

* `app.py`：**應用展示層**。主程式，負責 Streamlit UI 介面渲染、接收使用者參數與繪製圖表。
* `data_loader.py`：**資料存取層**。負責 yfinance 資料下載、技術指標計算（RSI、MA20），並預留「基本面、技術面、籌碼面」多因子過濾器介面。
* `model_core.py`：**AI 邏輯運算層**。負責 LSTM 神經網路建構、無未來函數的資料前處理與訓練/預測迴圈。
* `backtest_core.py`：**回測引擎層**。負責逐日交易模擬、執行進出場邏輯與風控，並計算最終量化績效指標。
* `requirements.txt`：依賴套件列表。
* `cuda.py`、`apptest.py`：輔助或測試用檔案（如有）。

## 策略判斷範例 (雙重防護進出場)

本系統的交易邏輯不再單純依賴 AI，而是結合了技術面濾網以降低假訊號風險：

* **【買進條件】必須「同時滿足」以下三點，系統才會在今天收盤時買進：**
  1. AI 模型預測明日報酬率為正（動能看多）。
  2. 今日收盤價大於 20 日均線（布林通道中軌，確保處於順勢）。
  3. 通過多因子條件（目前實作為：今日成交量大於 5 日均量之爆量表態）。

* **【賣出條件】只要滿足「任一」條件，系統就會在今天收盤時賣出結算：**
  1. **停損/停利觸發**：持有期間帳面損益觸及設定的停損（如 -5%）或停利標準，無條件優先平倉。
  2. **AI 訊號轉弱**：模型預測明日報酬率轉為負值。
